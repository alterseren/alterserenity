Ось список основних команд Git, які можуть знадобитися для роботи з репозиторіями:

1. **Створення нового репозиторію**:
   ```bash
   git init
   ```

2. **Клонування існуючого репозиторію**:
   ```bash
   git clone <URL>
   ```

3. **Перевірка статусу репозиторію**:
   ```bash
   git status
   ```

4. **Додавання файлів до індексу**:
   ```bash
   git add <файл>
   git add .
   ```

5. **Створення коміту**:
   ```bash
   git commit -m "Опис змін"
   ```

6. **Перегляд історії комітів**:
   ```bash
   git log
   ```

7. **Оновлення локальної гілки з віддаленого репозиторію**:
   ```bash
   git pull
   ```

8. **Відправлення змін на віддалений репозиторій**:
   ```bash
   git push
   ```

9. **Створення нової гілки**:
   ```bash
   git branch <назва_гілки>
   ```

10. **Переключення на іншу гілку**:
    ```bash
    git checkout <назва_гілки>
    ```

11. **Об'єднання гілок**:
    ```bash
    git merge <назва_гілки>
    ```

12. **Видалення гілки**:
    ```bash
    git branch -d <назва_гілки>
    ```

13. **Скидання змін**:
    ```bash
    git reset --hard
    ```

14. **Відміна останнього коміту (без видалення змін)**:
    ```bash
    git reset --soft HEAD~1
    ```

Ці команди охоплюють базові дії, які зазвичай потрібні для роботи з Git. Якщо потрібно більше деталей по будь-якій з команд, дай знати!
